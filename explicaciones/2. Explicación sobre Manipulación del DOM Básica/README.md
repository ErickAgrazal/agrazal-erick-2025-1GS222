# Manipulación del DOM Básica

Este ejemplo profundiza en la manipulación del DOM y la respuesta a eventos de usuario.

## Implementación

### HTML
- Muestra un nombre y un contenedor vacío `<span>` que se llena dinámicamente.
- Incluye un botón para interacción.

### CSS
- Estilos mejorados para el botón con efectos de borde y fondo al pasar el mouse (`hover`).
- Clase `.red` para cambios de estilo dinámicos.

### JavaScript
- **Template Strings**: Construcción de mensajes dinámicos combinando texto y variables.
- **Inyección de Contenido**: Asignación de texto al DOM mediante `textContent`.
- **Manejo de Eventos**: El botón alterna la clase `.red` en el texto, demostrando cómo JavaScript puede modificar estilos en tiempo real sin recargar la página.
- **Funciones**: Definición básica de funciones (`normal`) para lógica reutilizable.
