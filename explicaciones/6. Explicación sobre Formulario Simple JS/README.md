# Formulario Simple JS

Variante simplificada del manejo de formularios para reforzar conceptos básicos.

## Implementación

### HTML
- Formulario básico con etiqueta, input y botón "Guardar".
- Contenedor `div#response`.

### CSS
- Tema oscuro.
- Estilos de botón con hover (borde rojo).

### JavaScript
- **Captura de Datos**: Escucha el evento `submit`, previene el comportamiento por defecto y captura el valor del input.
- **Salida de Texto**: Muestra el resultado usando `textContent`, reemplazando el contenido previo. Es ideal para mostrar un único resultado o estado actual.
